# Sakura Astro 静态博客

**特性**：日/夜间模式 · 全屏封面 · 打字机效果 · 全文搜索 · 代码高亮 (200+ 语言) · KaTeX 数学公式 · RSS · Giscus 评论 · 友链 · 追番 · Steam 游戏库

## 快速开始

```bash
npm install
npm run dev     # http://localhost:4321
npm run build   # 产出 dist/
```

## 配置

编辑 `src/config/site.ts`，填写站点信息即可。
